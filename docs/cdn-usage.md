# CDN Usage

## Versioned URLs (Recommended)

```html
<script src="https://cdn.wokki20.nl/content/jspt-v2.0.4/jspt.min.js"></script>
<link rel="stylesheet" href="https://cdn.wokki20.nl/content/jspt-v2.0.4/jspt.css">
```

## Dynamic URLs (Latest)

```html
<script src="https://cdn.wokki20.nl/dynamic/jspt/jspt.js"></script>
<link rel="stylesheet" href="https://cdn.wokki20.nl/dynamic/jspt/jspt.css">
```

| Use Case | Recommended URL |
|----------|----------------|
| Production | versioned minified JS |
| Testing/Development | dynamic JS |
| ES6 Projects | ES Module |
